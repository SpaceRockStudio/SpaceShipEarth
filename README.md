# SpaceRockFramework
A game framework for Unity.

## Current Features

- Camera Fitter
- Standard GameObjects
  - 9-Slice
- Utility Classes
  - Sprite
  - String
  - Tilemap
- Tweener
  - float, double, 
  - int, uint, 
  - long, ulong
  - BigInteger
  - Vector2, Vector3, Vector4, 
  - Quaternion
  - Transform
    - Position
    - Rotation (EulerAngle & Quaternions)
    - Scale
